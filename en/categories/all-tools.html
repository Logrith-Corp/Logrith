The Complete Free Online Tool Directory

Logrith is a single destination for all the browser-based utilities a professional, student, or creator needs. Instead of bookmarking a dozen different sites — one for image conversion, another for PDF manipulation, a third for JSON formatting — Logrith consolidates 160+ tools under one consistent, ad-free interface.

Every tool on this page runs 100% client-side. The Image Tools use the HTML5 Canvas API to process and re-encode images without any server involvement. The PDF Tools use PDF-Lib and PDF.js, both mature JavaScript libraries that run entirely in-browser. The Developer Tools leverage the Web Crypto API, native JSON parsing, and JavaScript's built-in RegExp engine. The Text Tools use pure JavaScript string operations. The Utility Tools use QRCode.js for QR generation and standard mathematical formulas for calculators.

The roadmap for Logrith includes audio tools (MP3 cutter, audio merge, format converter), additional image editing (watermark, text overlay, brightness/contrast), more developer tools (CSS/JS minifiers, HTML formatter, XML validator), extended text utilities (Markdown converter, email extractor, keyword density), and additional calculators (discount, fuel cost, date difference). All tools follow the same no-login, no-upload, instant-result philosophy that makes Logrith genuinely useful.

Frequently Asked Questions

Are all Logrith tools completely free?
Yes — no exceptions. No paid plan, no usage limits, no watermarks, no account required. Every tool listed on this page is free forever.
Do any of these tools upload my files to a server?
None. Every single tool processes data in your browser. This is verifiable — open Chrome DevTools Network tab on any tool page and you'll see zero upload requests when using the tools.
How many tools are currently live vs "coming soon"?
Currently 40+ tools are fully functional. The remaining are on the roadmap with "Coming Soon" labels. New tools are added regularly — check back or bookmark this page.
Can I suggest a tool that should be added?
Yes! The most-requested tools get prioritized. Tools that can run client-side (in the browser) without requiring a server are most likely to be implemented quickly.
Do the tools work on mobile?
All tools are fully mobile-responsive. Image tools support camera input on iOS and Android. PDF tools work on mobile browsers. Some complex tools (like batch processing) may be slower on low-end mobile devices.
Why do some tools require an internet connection even if they're "client-side"?
Some tools load JavaScript libraries from CDNs on first use — PDF-Lib, PDF.js, QRCode.js. After the library loads, the actual processing is local. Once a page has been loaded, most tools work offline. This is a performance tradeoff to keep page sizes small.
What browsers are supported?
Chrome 90+, Firefox 88+, Safari 14+, Edge 90+, and Brave. The tools use standard modern web APIs that all these browsers support. Internet Explorer is not supported.
How is Logrith different from similar sites like ILovePDF or Smallpdf?
The key difference is that ILovePDF and Smallpdf upload your files to their servers for processing. Logrith processes everything locally in your browser. For sensitive documents — financial records, contracts, personal photos — this privacy difference is significant.
Can I use Logrith tools for commercial projects?
Yes. There are no restrictions on commercial use. Convert product images, generate QR codes for marketing materials, format API data, calculate business metrics — all commercial use cases are welcome.
Is there an API for Logrith tools?
Not currently. Logrith is a browser-only tool platform. For programmatic/API access to similar functionality, consider the libraries Logrith uses internally — PDF-Lib (npm), Sharp (Node.js image processing), and the Web Crypto API are all available for direct use in your applications.
