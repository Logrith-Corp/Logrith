About Logrith Utility Tools

Utility tools are the workhorses of everyday digital tasks — from generating a QR code for a business card to calculating the exact EMI on a home loan. Logrith's utility tools are designed for speed and practicality, with zero friction between arriving at the page and getting your result.

The QR Code Generator covers seven different QR content types in one tool — URL/link, plain text, email (with subject and body), phone number (tel: URI), SMS with pre-filled message, WiFi credentials (SSID, password, security type), and full vCard contact information. All QR codes are generated using QRCode.js in the browser, download as PNG (for digital use) or SVG (for print-quality scalable output).

The Color Converter is particularly useful for designers and frontend developers who work across different tools with different color format requirements — Figma often uses HSL, CSS traditionally uses HEX, print design requires CMYK, and older graphics tools use RGB 0–255. The tool converts instantaneously between all four formats as you adjust any input, with a color picker, 40-swatch tint/shade palette, and common named color reference.

For Indian users, the GST Calculator supports all current GST slabs (5%, 12%, 18%, 28%) plus exempt (0%), with both GST-inclusive (where the entered price already includes tax) and GST-exclusive (where tax is added on top) modes. The breakdown table shows both CGST+SGST (for intra-state transactions) and IGST (for inter-state transactions), making it useful for both businesses and consumers.

The EMI Calculator uses the standard compound interest EMI formula (P × r × (1+r)^n / ((1+r)^n - 1)) and generates a complete amortization schedule showing how much of each payment goes toward principal versus interest over the loan tenure. This is invaluable for home loan planning, comparing different tenure options, and understanding the total cost of borrowing.

Frequently Asked Questions

Can I create a WiFi QR code that guests can scan to connect?
Yes! Select "WiFi" in the QR Generator, enter your WiFi network name (SSID), password, and select the security type (WPA2 for most modern routers). Scanning this QR code on Android and iOS devices automatically offers to connect to that network — no manual password typing needed.
What QR code size should I use for printing?
For printed materials, use 400px or 512px from the generator and download as SVG for print-quality output (vector graphics scale to any size without pixelation). The minimum scannable printed size is about 2×2 cm at standard print resolutions.
How do I convert Celsius to Fahrenheit?
Open the Unit Converter, select "Temperature" category, enter your value, and choose "Celsius" as the from unit. All temperature equivalents (Fahrenheit, Kelvin, Rankine) appear simultaneously. Formula: °F = (°C × 9/5) + 32.
What is the current GST rate in India for restaurants?
Non-AC restaurants: 5% GST (no ITC). AC restaurants or restaurants in starred hotels: 5% GST. Restaurants in hotels with room tariff above ₹7,500: 18% GST (with ITC). The GST Calculator supports all these rates — just enter the applicable percentage.
How accurate is the EMI calculator?
The EMI calculation uses the standard reducing balance formula identical to what banks use. Results match bank EMI calculators exactly when the same loan amount, interest rate, and tenure are entered. The only variable is whether your bank uses monthly or annual reducing balance — most use monthly, which this calculator uses.
Can I calculate color contrast for accessibility (WCAG)?
The color converter shows RGB values which you can use in accessibility contrast calculators. A dedicated WCAG contrast checker — which computes the luminance ratio between two colors and checks WCAG AA/AAA compliance — is on the development roadmap.
How do I calculate percentage increase from one year to the next?
Use the "Percentage Change (Increase/Decrease)" calculator in the Percentage Calculator. Enter the old value (e.g., last year's sales) and the new value (this year's sales). The tool shows the percentage change and whether it's an increase or decrease.
What is the HEX color code for white, black, and common web colors?
White: #FFFFFF, Black: #000000, Red: #FF0000, Green: #00FF00, Blue: #0000FF, Yellow: #FFFF00, Cyan: #00FFFF, Magenta: #FF00FF, Gray: #808080. The Color Converter includes a named colors palette for quick access to these and 20+ other common colors.
How many MB is 1 GB?
In the SI standard (used by storage manufacturers): 1 GB = 1,000 MB. In the binary standard (used by operating systems): 1 GiB (gibibyte) = 1,024 MiB. This discrepancy is why a "1 TB" hard drive shows as ~931 GB in Windows. The Unit Converter supports both standards (GB/MB and GiB/MiB).
Can I calculate age in months and days for a baby or infant?
Yes. The Age Calculator shows the complete breakdown including months and days remaining beyond complete years. For a 6-month-old, it would show "0 years, 6 months, N days". You can also calculate age as of any date in the past or future.
