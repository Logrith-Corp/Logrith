About Logrith Image Tools

Image processing is one of the most common tasks on the web, yet most online image tools either require account creation, impose file size limits, or secretly upload your photos to third-party servers. Logrith's image tools are fundamentally different — every operation runs inside your browser using the HTML5 Canvas API, which means your photos never leave your device.

The Image Converter handles the most common format conversions: JPG to PNG (for lossless quality or transparency support), PNG to JPG (to reduce file size for photos), PNG/JPG to WEBP (for modern web optimization), and any format to BMP or GIF. The conversion is lossless for PNG output and quality-controlled for JPG and WEBP outputs through a quality slider.

The Image Compressor uses re-encoding at a lower quality level to reduce file sizes. At 75–80% quality, most images look visually identical to the original but are 40–70% smaller — ideal for speeding up websites, reducing storage, or staying within email attachment limits. The tool supports batch compression of up to 10 images simultaneously.

For layout work, the Image Resize tool includes one-click presets for common sizes — HD 1920×1080, 720p, Open Graph (1200×630), profile photo (400×400) — and an aspect ratio lock to prevent distortion. The Image Crop tool supports drag-select cropping directly on the image preview, with aspect ratio presets (1:1 square, 16:9 landscape, 9:16 portrait) and exact pixel coordinate entry.

Frequently Asked Questions

Are these image tools safe for personal photos?
Yes, completely. Every image tool on Logrith processes your photos exclusively in your browser's memory. No image data is transmitted to any server. You can verify this by opening the Network tab in browser DevTools — zero requests are made after the page loads.
How do I convert JPG to WEBP for my website?
Open the Image Converter, upload your JPG file, select WEBP as output format and set quality to 80%, then click Convert. WEBP files are typically 25-35% smaller than equivalent JPGs, directly improving your site's Core Web Vitals and page load speed.
What's the best way to compress images for WordPress?
For WordPress, use the Image Compressor at 75-80% quality for photos, convert to WEBP format for modern browsers, and resize to your theme's actual display width before uploading — there's no benefit to uploading a 4000px image if your content column is 800px wide.
Can I resize multiple images to the same dimensions?
Currently the resize tool processes one image at a time. For bulk resizing, use the Image Compressor which processes up to 10 images simultaneously with a consistent quality setting. Bulk resize with consistent dimensions is on the development roadmap.
Why does my PNG become larger after converting to JPG?
This can happen if the original PNG contained large flat color areas or gradients, which compress very efficiently in PNG's lossless format. JPG's lossy compression is optimized for photographic content with continuous tone variation, not for graphics or text with sharp edges.
What's the difference between crop and resize?
Resize changes the overall dimensions of the image — it scales all content up or down. Crop removes pixels from the edges, selecting only a portion of the original image at its original quality. Use resize to scale an image smaller; use crop to change composition.
What image sizes are recommended for social media?
Instagram square post: 1080×1080. Instagram landscape: 1080×566. Twitter/X card: 1200×675. Facebook post: 1200×630. LinkedIn post: 1200×627. Pinterest pin: 1000×1500. Use the Image Resize presets to hit these sizes quickly.
Does image rotation affect quality?
In Logrith's rotate tool, rotation is done through canvas transform operations and the output is re-encoded as PNG, which is lossless. There is no quality degradation from rotation in this tool. If you rotate a JPG, the output will be PNG — convert back to JPG afterward if needed.
What is the maximum file size for image tools?
Each image tool has a 20MB per-file limit. For most use cases this is more than sufficient — a typical 12 megapixel camera photo is 3-5MB. Very large RAW images from professional cameras may exceed this limit and should be pre-processed with desktop software.
Can I use these tools on iPhone/iOS?
Yes. All tools work on iOS Safari and Chrome. The file picker on iOS supports selecting from Photos, Files, and iCloud. Note that HEIC format (default iPhone photo format) is not directly supported by browser Canvas API — convert HEIC to JPG first using your phone's built-in share options.
