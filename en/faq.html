Frequently Asked Questions

Everything you need to know about Logrith — how it works, what's safe, and what's coming next.

🌐 General
What is Logrith?
Logrith is a free online tool hub with 160+ browser-based utilities for image editing, PDF management, developer workflows, text processing, and everyday calculations. Every tool runs 100% in your browser — no server upload, no account, no install required.
Is Logrith really free? Are there hidden costs?
Completely free, no exceptions. No premium tier, no usage credits, no watermarks on downloads, no account required. The platform is supported by non-intrusive advertising, not by charging users.
How many tools does Logrith have?
Logrith currently has 40+ fully functional tools live, with a total roadmap of 160+ tools. New tools are added regularly based on user demand. Categories include Image, PDF, Developer, Text, Utility, Audio (coming soon), and File Tools (coming soon).
Do I need to create an account or log in?
No. Zero registration, zero login, zero personal information required. Every tool is immediately accessible. There is no sign-up form anywhere on Logrith.
Which browsers does Logrith support?
All modern browsers — Chrome 90+, Firefox 88+, Safari 14+, Edge 90+, and Brave. Both desktop and mobile versions of these browsers are supported. Internet Explorer is not supported as it lacks the modern web APIs Logrith tools rely on.
🔒 Privacy & Safety
Are my files uploaded to Logrith's servers?
No. This is the most important thing to understand about Logrith: all file processing happens entirely in your browser using JavaScript APIs. Your images, PDFs, and documents are never transmitted to any server. You can verify this yourself by opening Chrome DevTools → Network tab and using any tool — you'll see zero file upload requests.
Is it safe to process sensitive documents on Logrith?
Yes. Since files never leave your device, Logrith is safe for sensitive documents — contracts, financial statements, personal photos, confidential business files. The processing happens in your browser's sandboxed JavaScript environment and the data is never accessible to us.
Does Logrith collect any personal data?
Logrith does not collect, store, or process any personal data or file contents. We use standard anonymous analytics (page views, general usage patterns) to understand which tools are most used. No individual user data, no file contents, no search queries are ever stored. See our Privacy Policy for full details.
Can I use Logrith offline?
Most tools work offline after the page has loaded once, since all processing is local. Some tools load JavaScript libraries from CDN (PDF-Lib, QRCode.js) which require an initial internet connection. After the library loads, the actual processing is offline-capable.
🖼️ Image Tools
How do I convert JPG to PNG on Logrith?
Open the Image Converter tool, upload your JPG file (drag & drop or click to browse), select PNG as the output format, then click Convert. The download starts immediately. You can also batch convert multiple files at once.
How much can I compress an image without losing quality?
At 75–80% quality, JPG and WEBP files are typically 40–70% smaller with no visible difference to the human eye. The Image Compressor shows you the exact before/after file sizes and percentage saved. PNG compression is lossless — it removes metadata without affecting visual quality.
What image formats are supported?
Logrith Image Converter supports JPG/JPEG, PNG, WEBP, BMP, and GIF. For output, you can convert to JPG, PNG, WEBP, or BMP. HEIC (iPhone format) is not directly supported by browser Canvas API — convert HEIC to JPG on your phone first using the built-in share options.
What is WEBP and why should I use it?
WEBP is a modern image format developed by Google. It produces files 25–34% smaller than equivalent JPG or PNG at the same visual quality. Using WEBP on your website directly improves Google PageSpeed scores and Core Web Vitals. All modern browsers support WEBP.
📄 PDF Tools
How do I merge multiple PDF files?
Open PDF Merge, upload all your PDF files, drag them to set the order if needed, then click Merge PDFs. The tool uses PDF-Lib running in your browser to combine the files. Download the merged PDF instantly. No file size limits beyond available browser memory.
Can I extract specific pages from a PDF?
Yes. In PDF Split, select "Extract Page Range" and enter ranges like "1-5, 8, 10-12". The tool creates a new PDF with exactly those pages. You can also split a PDF into individual single-page PDFs using the "Split All Pages" option.
Will PDF processing reduce document quality?
No. PDF-Lib copies page objects directly from source files without re-encoding. Text remains selectable, images retain original resolution, and all formatting is preserved. Merging, splitting, and rotating are metadata operations that don't affect content quality.
What is the maximum PDF file size?
Each PDF has a 20MB individual file limit. Total combined size for merge operations depends on available browser memory (typically 100–200MB on desktop). For very large PDFs, we recommend splitting them into smaller chunks first.
🛠️ Developer Tools
Is the password generator cryptographically secure?
Yes. Logrith's Password Generator uses crypto.getRandomValues() — the Web Crypto API — which provides cryptographically secure randomness equivalent to what operating systems and security software use. It is not the weak Math.random() used by many generator sites.
Can I hash files, not just text?
Yes. The Hash Generator supports files up to 100MB. It uses SubtleCrypto.digest() to compute SHA-1, SHA-256, SHA-384, and SHA-512 hashes. File hashing is useful for verifying download integrity — compare the hash with what the software publisher provides.
Does the JSON formatter work on large files?
The JSON formatter works on files up to about 50–100MB depending on your device's available memory. For very large JSON files, use the command-line tool jq instead. For typical API responses and config files, the formatter handles any size without issues.
⚙️ Technical
A tool isn't working — what should I do?
1) Try refreshing the page. 2) Clear browser cache (Ctrl+Shift+R). 3) Try Chrome or Firefox if on another browser. 4) Check that your file is within the size limit. 5) If the issue persists, please report it on our Contact page with the tool name and what happened — we fix bugs within 48 hours.
Why do some tools need internet even though they're "client-side"?
Some tools load JavaScript libraries (PDF-Lib, PDF.js, QRCode.js) from CDN on first use. After the library downloads, the actual processing is local. Once you've visited a tool page, most tools work offline due to browser caching. Pure JavaScript tools (text tools, calculators, JSON formatter) work fully offline from the first visit.
Can I use Logrith tools programmatically via API?
Logrith is a browser-only platform — there is no public API. For programmatic access, use the same libraries we use: PDF-Lib (npm), Sharp (Node.js image processing), and the Web Crypto API directly in your JavaScript code. An API is on the long-term roadmap.
