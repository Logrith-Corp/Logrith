What is a Hash?

A hash is a unique fixed-length string generated from input data using cryptographic algorithms. Same input always produces same hash.

Hash Types

  • MD5: 128-bit hash (deprecated for security)
  • SHA1: 160-bit hash (deprecated)
  • SHA256: 256-bit secure hash
  • SHA512: 512-bit strongest hash

Uses

  • Data integrity verification
  • Password storage
  • File verification
  • Blockchain applications
  • Security authentication
