Frequently Asked Questions

Everything you need to know about Logrith

What is Logrith and what tools does it offer?
Logrith is a free online tool hub offering 160+ browser-based utilities for everyday digital tasks. It includes image tools (converter, compressor, resize, crop, rotate), PDF tools (merge, split, convert), developer tools (JSON formatter, Base64, URL encoder, hash generator, UUID generator), text tools (word counter, case converter, text diff), and utility tools (QR generator, color converter, calculators). All tools run 100% in your browser — no server upload, no login required.
Is Logrith completely free to use?
Yes, Logrith is 100% free with absolutely no hidden charges. There are no premium plans, no usage credits, no watermarks on output files, and no account required. Every single tool on the platform is available to all users at no cost, forever.
Are my files safe when using Logrith tools?
Completely safe. Logrith processes all files directly in your browser using JavaScript APIs (Canvas API, Web Crypto API, PDF-Lib). Your images, PDFs, and documents are never uploaded to any server. You can even use most tools offline after the page has loaded. No file data ever leaves your device.
Do I need to create an account or log in?
No. Logrith requires zero registration, zero login, and zero personal information. Simply open any tool URL and start using it immediately. There is no sign-up form, no email verification, and no account management of any kind.
Does Logrith work on mobile phones?
Yes, all Logrith tools are fully mobile-responsive and tested on iOS and Android browsers. The interface adapts to any screen size. Image tools support camera input on mobile, and all output files are directly downloadable to your phone's storage.
What is the maximum file size I can upload?
Most tools support up to 20MB per file. The file hash tool supports up to 100MB. These limits ensure smooth performance on all devices including budget smartphones. For very large files, we recommend using desktop hardware for the best experience.
Can I convert JPG to PNG for free on Logrith?
Yes! Open the Image Converter tool, upload your JPG file, select PNG as the output format, and click Convert. The conversion happens instantly in your browser. You can also batch convert multiple images at once, and choose quality levels for lossy formats like JPG and WEBP.
How do I merge PDF files online for free?
Open the PDF Merge tool on Logrith, upload all the PDF files you want to combine, drag them to reorder if needed, then click Merge. The tool uses PDF-Lib to combine the files entirely in your browser. Download the merged PDF instantly — no email required, no file size watermark.
Which browsers are supported?
Logrith works on all modern browsers — Chrome, Firefox, Safari, Edge, and Brave — on both desktop and mobile. The tools use standard web APIs that all modern browsers have supported for years. Internet Explorer is not supported.
Will Logrith remain free? Are there ads?
Logrith is committed to remaining free for all users. The platform may display minimal, non-intrusive advertisements to cover operational costs, but these never interrupt tool usage, never track your files, and never affect the quality of output. No paywalls, no forced upgrades — ever.
