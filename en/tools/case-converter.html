Frequently Asked Questions

What is camelCase and when to use it?
camelCase starts words with lowercase, then uppercase (helloWorld). Used for JavaScript variables, functions, Python methods. First word lowercase distinguishes from PascalCase.
PascalCase vs camelCase — what's the difference?
PascalCase capitalizes EVERY word including first (HelloWorld). Used for classes, components (React), TypeScript types. camelCase (helloWorld) for variables/functions.
snake_case vs kebab-case — which for what?
snake_case (hello_world) → Python variables, database columns, Ruby. kebab-case (hello-world) → CSS classes, HTML data attributes, URL slugs, file names.
What is Title Case vs Sentence case?
Title Case: Every Major Word Capitalized (Hello World). Sentence case: Only First Word (Hello world). Title for headlines, Sentence for body text.
Can I convert code variable names?
Yes! Convert between camelCase, snake_case, kebab-case, PascalCase instantly. Perfect for refactoring across languages/frameworks.
