Frequently Asked Questions

What is camelCase?
camelCase is a naming convention where the first word is lowercase and each subsequent word starts with an uppercase letter, with no spaces (e.g., helloWorld, myVariableName). It's widely used in JavaScript and many programming languages.
What is the difference between PascalCase and camelCase?
PascalCase (also called UpperCamelCase) starts with an uppercase letter — every word is capitalized (e.g., HelloWorld). camelCase starts with a lowercase letter (e.g., helloWorld). PascalCase is common for class names and components.
When should I use snake_case vs kebab-case?
snake_case uses underscores and is common in Python variable names, database column names. kebab-case uses hyphens and is common in CSS class names, HTML attributes, and URL slugs.
What is Sentence case?
Sentence case capitalizes only the first letter of the first word, with the rest in lowercase — just like a regular sentence. It's used in article titles, UI labels, and general writing.
