Frequently Asked Questions

What is Base64 encoding?
Base64 is an encoding scheme that converts binary data into a set of 64 ASCII characters (A–Z, a–z, 0–9, +, /). It's commonly used to embed binary data like images in text-based formats such as HTML, JSON, or email.
What is URL-safe Base64?
URL-safe Base64 replaces + with - and / with _ so the encoded string can be used safely in URLs without percent-encoding. It's used in JWT tokens and other web contexts.
Can I encode images to Base64?
Yes! Use the "File → Base64" tab and upload an image. The tool converts it to a Base64 data URI (e.g., data:image/png;base64,...) which you can embed directly in HTML or CSS.
How much does Base64 increase file size?
Base64 encoding increases the data size by approximately 33%. So a 100KB image becomes about 133KB when Base64-encoded. This is a trade-off for embedding convenience.
