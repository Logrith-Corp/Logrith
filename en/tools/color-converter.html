Frequently Asked Questions

What is the difference between HEX and RGB colors?
Both represent the same color. RGB uses three decimal values (0–255) for Red, Green, Blue. HEX is simply RGB values written in hexadecimal notation (#RRGGBB). For example, rgb(255,0,0) = #FF0000 (red).
What is HSL color format?
HSL stands for Hue, Saturation, Lightness. Hue is the color type (0–360°), Saturation is color intensity (0–100%), and Lightness is the brightness (0% = black, 100% = white, 50% = pure color). HSL is more intuitive for design work.
What is CMYK and when is it used?
CMYK (Cyan, Magenta, Yellow, Key/Black) is a subtractive color model used in physical printing. Unlike RGB (additive, for screens), CMYK represents how ink mixes on paper. Use CMYK values when sending designs to print.
