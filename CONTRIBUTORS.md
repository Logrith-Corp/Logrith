### How to Contribute

We welcome contributions from the community! Here's how you can help:

#### Reporting Issues

1. Go to [GitHub Issues](https://github.com/Logrith-Corp/Logrith/issues)
2. Click "New Issue"
3. Provide detailed description
4. Include screenshots/error messages if applicable

#### Submitting Code

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test thoroughly on multiple browsers
5. Submit a Pull Request with description

#### Code Style Guidelines

- Use semantic HTML5
- Write accessible CSS3
- Keep JavaScript vanilla (no frameworks)
- Follow existing code patterns
- Add comments for complex logic
- Test on mobile devices

### Recognition

All contributors will be recognized in:
- This file (CONTRIBUTORS.md)
- GitHub contributions graph
- Release notes for related versions

## 📚 Resources for Contributors

### Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Logrith-Corp/Logrith.git
   cd Logrith
   ```

2. **Start a local server:**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Or use Live Server in VS Code
   # Or use Node.js http-server
   npx http-server
   ```

3. **Open in browser:**
   ```
   http://localhost:8000/en/
   ```

### Project Structure

```
Logrith/
├── en/                    # Main content
│   ├── tools/            # Individual tool pages
│   ├── categories/       # Category pages
│   ├── pages/            # Static pages (contact, privacy, etc)
│   └── components/       # Reusable components (navbar, footer)
├── assets/               # CSS, JS, SVG, images
├── README.md             # Main documentation
├── LICENSE               # MIT License
├── PRIVACY.md            # Privacy Policy
├── SECURITY.md           # Security Policy
└── CONTRIBUTORS.md       # This file
```

### Development Tips

- No build tools needed - edit HTML/CSS/JS directly
- Use browser DevTools for debugging
- Test on real mobile devices when possible
- Check browser compatibility (all modern browsers supported)
- Add comments for complex JavaScript logic
